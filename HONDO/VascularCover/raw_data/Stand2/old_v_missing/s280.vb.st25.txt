This is a modified version of the Hondo-Slave Lake Reference Stand
vascular and bryoid August 1980 survey data.  The modification was
done by GHLR's critical analyses of important species estimated
cover % values, especially in relationship to 1981 and 1982 values.
Original and modified versions of this survey data are in *FS-saved
files of the same name; only the modified data file has this header.
In some cases, a few unrelated modifications were also made to
correct errors in the original data.
 
Since this modified survey data is now in fixed format, a new SPSS
control card file was created to account for this; it has the same
file name as the older version--use the control card file with the
same *FS version number as this data file.
